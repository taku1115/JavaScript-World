### JavaScriptの評価すべき点
- 第一級関数とクロージャをサポート
- プロパティを随時追加できる柔軟なオブジェクト
- 表現力の高いリテラル記法
### 変数宣言 var の問題点
- １：再宣言が可能 
- ２：変数の参照が巻き上げられる
- ３：スコープ単位が関数
- 関数宣言文にも同様のデメリットあり（関数式推奨）
### セミコロンの例外規則
- 基本的には文の末尾に必要
- しかし、{}ブロックで終わる場合にはつけない
- 式の場合は{}ブロックを最終的に変数に代入しているためセミコロンをつける
